# Project-Wide Instructions

This repository uses [style.md](style.md) as the authoritative site style guide.

Apply it to all work in this project, especially when editing pages, writing copy, or updating documentation.

## Default Rules

- Gaeilge first, then English.
- Keep Irish as the primary language wherever bilingual content appears.
- Preserve the retro, calm, accessible, PowerPC-friendly feel.
- Keep markup lean and compatible with older browsers where practical.
- Keep visible accessibility affordances: contrast, focus states, language markup, and readable spacing.

## When Editing

- Check [style.md](style.md) before making content or design changes.
- Follow the existing project instructions in [.instructions.md](.instructions.md) as well.
- Prefer small, consistent updates that match the established site voice.
