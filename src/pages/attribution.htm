Fonts

Atkinson Hyperlegible Next — designed by the Braille Institute of America for maximum legibility, including for low-vision readers. Used as the primary body and UI font. SIL Open Font License 1.1
Inter — by Rasmus Andersson. Fallback / general UI font where Atkinson isn't appropriate. SIL OFL 1.1
Fraunces — by Undercase Type. Reserved for display headings on long-form pages. SIL OFL 1.1
JetBrains Mono — by JetBrains. Used for code samples and terminal snippets. SIL OFL 1.1

Badges & web-button graphics

Badges & web buttons — "88x31" web buttons from the web-1.0 era, sourced from 88x31.lol and An Lucas (Neocities). Classic 88×31 pixel graphics that have circulated freely across the web for decades.
"For The Badge" project-status badges (used on previous iterations of the site) — forthebadge.com.

Inspiration & references

The visual language of this site is a love letter to the late-1990s / early-2000s Mac web — iMac G3, iBook Clamshell, iBook G4, Mac OS 9.2, 10.0 Cheetah and 10.4 Tiger all sit somewhere in its DNA.
